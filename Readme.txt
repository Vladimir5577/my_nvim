Nvim key binding.
-------------------

Lsp:
    Shift + k  ---> hover
    g+d        ---> go to definition

Autocompletion:
    Tab ---> confirm
    C-k ---> previous suggestion
    C-j ---> next suggestion
    C-b ---> scroll_docs(-4),
    C-f ---> scroll_docs(4),
    C-Space ---> show completion suggestions
    C-e ---> close completion window


Telescope:
    Space ff  ---> find file
    Space fg  ---> find grep (text in project)

Neo tree toggle
    Ctrl + n

Comment:
    gcc ---> line Comment
    gbc ---> block comment

Format code (comform plugin)
    Space + l

Git blame:
    :BlameToggle
